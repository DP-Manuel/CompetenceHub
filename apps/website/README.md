# Website

Astro workspace for the public website.

## Current Preview

The first page is a low-fidelity homepage preview with two variants:

- Variant A: seriös-corporate
- Variant B: modern-dynamic

Both variants use the same B2B-first content foundation and a shared contact form section.

## Local Commands

Use the portable Node.js runtime from the repository root:

```powershell
$env:PATH = "$PWD\tools\node-v22.16.0-win-x64;$env:PATH"
cd apps\website
npm install
npm run dev
```
